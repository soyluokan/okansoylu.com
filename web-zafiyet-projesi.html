Proje Hakkında

Bu proje, BTK-USOM'da gerçekleştirdiğim staj sürecinde, web güvenliği eğitimi amacıyla geliştirdiğim bir çalışmadır. Staj sürecimde edindiğim siber güvenlik bilgi ve deneyimlerini, özellikle web uygulama güvenliği alanındaki öğrendiklerimi, interaktif bir şekilde göstermek için tasarladım.

Proje, üç temel web güvenlik açığını (XSS, SQL Injection ve OS Command Injection) içeren zafiyetli bir versiyon ve bu açıkların nasıl kapatılacağını gösteren güvenli bir versiyon olmak üzere iki ayrı uygulama içermektedir. Bu sayede, güvenlik açıklarının nasıl oluştuğunu ve nasıl önlenebileceğini pratik bir şekilde göstermeye çalıştım.

Özellikler

XSS (Cross-Site Scripting)

Kullanıcı girdilerinin doğrudan HTML içeriğine dönüştürülmesi sonucu oluşan XSS açıklarını test edebilir, zararlı JavaScript kodlarının nasıl çalıştığını ve nasıl engellendiğini gözlemleyebilirsiniz.

SQL Injection

Veritabanı sorgularına kötü niyetli kodların enjekte edilmesiyle oluşan SQL Injection açıklarını test edebilir, veritabanından hassas bilgilerin nasıl sızdırılabildiğini ve güvenli sorgulamanın nasıl yapıldığını öğrenebilirsiniz.

OS Command Injection

İşletim sistemi komutlarına zararlı kodların eklenmesiyle oluşan Command Injection açıklarını test edebilir, sistemde yetkisiz komutların nasıl çalıştırılabildiğini ve güvenli komut çalıştırmanın nasıl yapıldığını inceleyebilirsiniz.

Güvenlik Açıkları ve Çözümleri

XSS (Cross-Site Scripting)

Zafiyetli Versiyon: Kullanıcı girdisi doğrudan ekrana yazdırılıyor:
$output .= "

" . $xss . "

";

Güvenli Versiyon: htmlspecialchars() fonksiyonu ile karakterler dönüştürülüyor:
htmlspecialchars($xss, ENT_QUOTES, 'UTF-8')

SQL Injection

Zafiyetli Versiyon: SQL sorgusu doğrudan kullanıcı girdisiyle oluşturuluyor:
$sql_query = "SELECT * FROM users WHERE id = '$sql'";

Güvenli Versiyon: Prepared statements kullanılıyor:
$stmt = $conn->prepare("SELECT * FROM users WHERE id = ?");

OS Command Injection

Zafiyetli Versiyon: Komut doğrudan çalıştırılıyor:
$command_output = shell_exec($command);

Güvenli Versiyon: IP adresi doğrulanıyor ve escapeshellarg() kullanılıyor:
if (filter_var($ipAddress, FILTER_VALIDATE_IP)) {
$command = 'ping ' . escapeshellarg($ipAddress);
